# buffer-crypto-closure.txt — symbol contract for the buffer consumer (crypto-only profile).
#
# Provenance: the deduped, sorted set of raw libcrypto functions called by the static-inline
# wrapper closure in buffer's buffer-crypto/src/nativeInterop/cinterop/boringsslcrypto.def
# (the canonical cryptoOnly consumer; 36 bcl_* wrappers over the 75 functions below).
# Consumed by scripts/validate-artifacts.sh check 7, which asserts every entry is DEFINED
# (T/W) in the bundle's extracted libcrypto.a — so a canonical-pin move can never silently
# drop a symbol buffer links against.
#
# Macro/enum-only tokens the wrappers use are deliberately EXCLUDED (they never appear in
# nm output): EVP_AEAD_DEFAULT_TAG_LENGTH (macro), the NID_* curve ids (defines),
# POINT_CONVERSION_UNCOMPRESSED (enum). libc calls (memcpy) are likewise excluded.
# Every entry below was verified as a T-defined function in a libcrypto.a built at the
# canonical catalog pin.
#
